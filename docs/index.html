SmartMeter Mobile App

Bluetooth接続でCEM6シリーズ電力メーターの情報取得、
42パターンの誤結線判定、計測試験をAndroid端末で実現

主要機能

メーター情報取得

Bluetooth経由でCEM6メーターの基本情報、計測値、瞬時値を取得。IR01/IR02/IR06テレグラムに対応。

  • メーターID、製造番号
  • 正方向/逆方向有効電力量
  • 瞬時電力・電流
  • 計測日時履歴

誤結線判定

42パターンの異常検知アルゴリズムで結線状態を自動判定。IR91テレグラムで現在値を取得。

  • 電圧系誤結線（1-21番）
  • 電流系誤結線（22-35番）
  • 混合誤結線（36-42番）
  • 推奨対処法の表示

計測試験

自動・手動モードで計測値を時系列記録。繰り越し対応の差分計算機能付き。

  • タイマー自動測定（10秒-60分）
  • 手動測定モード
  • 最大3回までの記録
  • 経過時間・差分表示

Excelエクスポート

3種類のフォーマットでデータをExcel出力。既存ファイルへの追記や重複チェックに対応。

  • Format A（きんでん社用）
  • Format B（大崎標準）
  • Format C（スパイダープラス連携）
  • 追加フィールド入力対応

Android端末

  • Android 5.0 (Lollipop) 以上
  • 推奨: Android 13.0以上
  • Bluetooth 2.0以上（SPP対応）
  • ストレージ: 100MB以上の空き容量

必要なハードウェア

  • TP-BT Bluetooth SPPアダプター
  • CEM6シリーズ電力メーター
  • IR01/IR02/IR06/IR91対応

必要な権限

  • Bluetooth接続・管理

インストール手順

1

APKファイルをダウンロード

上記のダウンロードリンクからAPKファイルをAndroid端末にダウンロードします。

2

提供元不明のアプリを許可

設定 → セキュリティ → 提供元不明のアプリ → ブラウザまたはファイルマネージャーを許可

3

APKファイルをタップしてインストール

ダウンロードフォルダからAPKファイルを選択し、「インストール」をタップします。

4

権限を許可

アプリ起動時にBluetooth、位置情報、ストレージの権限を全て許可してください。
