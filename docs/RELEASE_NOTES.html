実装済み機能

120 121

1. メーター情報取得機能

122

概要: Bluetoothで接続したスマートメーターから各種情報を取得

123
    124
  • 基本情報（IR01テレグラム）: メーターID、製造番号、製造年月、動作モード等
    • 125
  • 計測情報（IR02テレグラム）: 正方向/逆方向有効電力量（現在値、前回値、前々回値）
    • 126
  • 瞬時値（IR06テレグラム）: 瞬時電力、瞬時電流
    • 127
  • 取得データの画面表示とExcel出力
    • 128
129 130

2. 誤結線判定機能

131

概要: メーターの結線状態を判定し、42パターンの誤結線を検出

132
    133
  • 判定パラメータ入力（定格電圧、定格電流、しきい値等）
    • 134
  • IR91テレグラムでの現在値取得（電圧、電流、進み・遅れ角度）
    • 135
  • 42パターンの自動判定（電圧系、電流系、混合）
    • 136
  • 判定結果の詳細表示と推奨対処法
    • 137
138 139

3. 計測試験機能

140

概要: メーターの計測値を時系列で記録し、誤差を計算

141
    142
  • 自動測定モード（タイマー設定: 10秒-60分）
    • 143
  • 手動測定モード
    • 144
  • 最大3回までの測定記録
    • 145
  • 繰り越し対応の差分計算（8桁）
    • 146
  • 経過時間のリアルタイム表示
    • 147
148 149

4. Excelエクスポート機能

150

概要: 取得したデータを3種類のExcel形式で出力

151
    152
  • Format A（基本情報）: メーターID、計測日、電力量、備考
    • 153
  • Format B（詳細情報）: Format A + 測定者名、場所、動作モード等
    • 154
  • Format C（検査記録）: Format B + 検査番号、検査者、承認者、合否判定
    • 155
  • 重複チェック、既存ファイルへの追記対応
    • 156
